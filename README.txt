Simple application that uses IQAir's AirVisual API and Google's API with phone GPS capabilities to determine air quality at the user's current location.

Delivers the US-based metric for air quality - a scale from 0-500 where:  
0-50 is Good  
51-100 is Moderate  
101-150 is Unhealthy for Sensitive Groups  
151-200 is Unhealthy  
201-300 is Very Unhealthy  
301-500 is Hazardous
